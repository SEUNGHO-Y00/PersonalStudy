## SFP

* An SFP (Small Form-factor Pluggable) module is a hot-swappable transceiver used in networking hardware like switches, routers, and firewalls. It acts as a modular adapter, allowing you to easily swap the type of cable—such as fiber optic or copper Ethernet—used for a specific port without replacing the entire device.

* These modules are highly versatile and are used to:
  - Connect Over Long Distances: By using fiber optic modules, networks can transmit data across campuses or even up to 100 km, far exceeding the 100-meter limit of standard copper cables.
  - Provide High-Speed Uplinks: SFPs are frequently used to link data closets or aggregate remote server rooms back to a central network backbone.
  - Change Cable Mediums: If a port requires copper, an RJ45 SFP module can be inserted. If it requires fiber, you simply swap it for an LC or SC fiber optic module.
  - Scale and Upgrade Networks: As bandwidth needs grow, you can upgrade from standard 1G SFPs to 10G SFP+ or 25G SFP28 modules simply by swapping the transceiver in the same port.

* Cisco GLC-SX-MMD : The Cisco GLC-SX-MMD is a 1000BASE-SX Small Form-Factor Pluggable (SFP) transceiver module. It provides 1 Gbps connectivity over multi-mode fiber (MMF) up to 550 meters, supporting Digital Optical Monitoring (DOM) and an 850nm wavelength.
